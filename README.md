# Secure Password Generator

Консольная утилита на Python для генерации криптографически стойких паролей. Скрипт не только создает случайные последовательности заданной длины, но и гарантирует соблюдение строгих политик безопасности, а также проверяет результат по словарю скомпрометированных паролей.

## Особенности
* **Криптографическая безопасность:** Использование модуля `secrets` (генератор псевдослучайных чисел, криптографически стойкий для управления секретами) вместо стандартного `random`.
* **Строгие правила (Password Policies):** Гарантированное наличие минимум одной строчной буквы, одной заглавной буквы, одной цифры и одного спецсимвола.
* **Защита от словарных атак:** Проверка сгенерированного пароля по блэклисту (файлу с часто используемыми паролями). Если пароль содержит популярную подстроку, он перегенерируется.
* **Гибкий CLI-интерфейс:** Настройка длины, количества паролей и перенаправление вывода в файл через аргументы командной строки (на базе `argparse`).

## Технологии
* Python 3.x
* Standard Library: `secrets`, `argparse`, `string`, `sys`

## Установка и запуск

1. Склонируйте репозиторий:
   ```bash
   git clone [https://github.com/ВАШ_НИК/password-generator.git](https://github.com/ВАШ_НИК/password-generator.git)
   cd password-generator
2. Убедитесь, что файл со словарем (10-char-common-passwords.txt) находится в директории со скриптом.
3. Запустите скрипт через командную строку:
   ```bash
   python Passwords.py [аргументы]

## Использование

Скрипт работает через интерфейс командной строки (CLI) и использует только стандартные библиотеки Python.
**Базовый синтаксис**
   bash
   python Passwords.py [-h] [-l LENGTH] [-c COUNT] [-w WRITE] [-d DICT]

## Примеры запуска

### 1. Быстрая генерация (настройки по умолчанию)
Создаст 1 надежный пароль длиной 16 символов и выведет его в терминал:
bash
python Passwords.py

### 2. Множественная генерация с заданной длиной
Сгенерирует 5 паролей, каждый длиной по 20 символов:
bash
python Passwords.py -l 20 -c 5

### 3. Сохранение результата в текстовый файл
Создаст 10 паролей (длиной 12 символов) и запишет результат в файл `my_passwords.txt`. В консоль будет выведено только сообщение об успешной записи:
bash
python Passwords.py --length 12 --count 10 --write my_passwords.txt

### 4. Использование кастомного словаря для проверки
Если ваш файл с популярными паролями называется иначе или лежит в другой папке:
bash
python Passwords.py -d ./wordlists/custom_blacklist.txt

### Доступные аргументы (Флаги)

| Короткий флаг | Полный флаг | Описание | Значение по умолчанию |
| :--- | :--- | :--- | :--- |
| `-h` | `--help` | Вызов встроенной справки с описанием всех команд. | - |
| `-l` | `--length` | Длина генерируемого пароля (минимальное значение — 4). | `16` |
| `-c` | `--count` | Количество паролей для генерации за один запуск. | `1` |
| `-w` | `--write` | Путь к файлу для экспорта результатов. Если не передан, пароли печатаются в `stdout`. | Вывод в консоль |
| `-d` | `--dict` | Путь к файлу-блэклисту для проверки паролей на уязвимость. | `10-char-common-passwords.txt` |
